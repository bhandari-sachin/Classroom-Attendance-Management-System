## Definition of Done
A task is considered **Done** when:
- All related tests are completed and passing
- Bugs are fixed and verified
- Code is pushed to the repository
- Code coverage meets the required threshold (≥ 80%)
- Jenkins pipeline runs successfully
- SonarQube analysis shows acceptable quality (A rating if possible)
- Documentation is updated
- Feature is validated through user acceptance testing
